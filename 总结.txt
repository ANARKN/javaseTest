ES6知识点
var, let, 和 const 是JavaScript中用于声明变量的关键字，它们的区别如下：

var 声明的变量是全局变量或函数级变量（如果在函数内部声明的话），并且它们具有变量提升的特性，也就是说，无论变量在代码中的位置如何，它们都会在代码执行之前被声明。如果在同一作用域中多次声明同一个变量，后面的声明将覆盖前面的声明。

let 和 const 是块级作用域的，也就是说它们只在声明它们的代码块中有效。 let 和 const 声明的变量不存在变量提升的特性，所以在声明之前使用这些变量会抛出 ReferenceError 错误。如果在同一作用域中多次声明同一个变量，后面的声明也将抛出错误。

let 声明的变量可以被重新赋值，而 const 声明的变量不能被重新赋值。但是，对于对象和数组这样的引用类型，即使使用 const 声明，也可以修改其属性或元素的值。所以，使用 const 声明对象或数组时，不能将其重新赋值为新的对象或数组，但可以通过修改其属性或元素来更改它们的值。

因此，建议在需要修改变量值的情况下使用 let，在不需要修改变量值的情况下使用 const。避免使用 var。
